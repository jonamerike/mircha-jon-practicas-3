# Practicas

## Creación de carpeta local

```bash
mkdir carpeta
cd carpeta
touch README.md
touch .gitignore
code .
```

## Inicialización de repositorio

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/jonamerike/mircha-jon-practicas-3.git
git push -u origin main
```
